//Input: 数组D[0..n]表示n个矩阵的维数
//Output: 给定维数的n个矩阵链相乘所需的最小乘法次数.T[1...n]用于获得乘法的最优顺序
for i ← 1 to n do M[i, i] ← 0
for d ← 1 to n − 1 do //diagonal count
	for i ← 1 to n − d do
		j ← i + d
		minval ←∞
		for k ← i to j − 1 do
			temp ← M[i, k] + M[k + 1, j] + D[i − 1] ∗ D[k] ∗ D[j]
			if temp < minval
			minval ← temp
			kmin ← k
		T[i, j] ← kmin
return M[1, n], T
